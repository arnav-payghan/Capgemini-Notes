> ZUSERABAP1_ALVLIST

``` 
DATA:
        i_fieldcat      TYPE slis_i_fieldcat_alv,   " Internal Table (Pre-Defined)
        wa_fieldcat     TYPE slis_fieldcat_alv,     " Work Area (Pre-Defined)
        ls_layout       TYPE slis_layout_alv.       
```

Why Cataloging - simple outputs of table.

How does this catalgoing work?

```
wa_fieldcat-tabname = 'IT_MARA'.                " Which table to refer to.
wa_fieldcat-fieldname = 'MATNR'.                " Field name of it.
wa_fieldcat-seltext = 'Material Number'.        " Name to give to the field. (m is for medium).
APPEND wa_fieldcat to i_fieldcat.
CLEAR wa_fieldcat.
```

> Note: [You need to do this for **ALL** the fields.]  
> Note: [Never use loops in cataloging. Prefer using subroutines because parameters can be used to substitute and change the values.]

How to create it?

```
PERFORM field_cat USING field_name sel_text 

FORM.
        wa_fieldcat-tabname = 'IT_MARA'.
        wa_fieldcat-fieldname = field_name
        wa_fieldcat-seltext = sel_text
        APPEND wa_fieldcat to i_fieldcat.
        CLEAR wa_fieldcat.
ENDFORM.
```
